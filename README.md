## 🌟 About

In a busy world, relationships often fade unintentionally. **KeenKeeper** is a modern relationship management app designed to help you stay connected with the people who matter most. It tracks your interactions, reminds you to reconnect, and gives you the context you need to keep every relationship meaningful.

---

## Core Features :

### 📞 Smart Quick Actions

Reach out instantly — **Call**, **Text**, or **Video** — with a single tap from any profile or dashboard card.

- One-tap access from the dashboard or contact profile
- Deep links to native phone, messaging & video apps
- Mobile-optimised tap targets for fast access on the go

### ⏳ Relationship Insights

See exactly how long it's been since your last interaction and act before the connection cools.

- Time-since-last-contact indicator on every profile
- Overdue reconnect warnings highlighted on the dashboard
- Custom check-in goal frequencies per person (weekly, monthly, etc.)

### 👤 Rich Friend Profiles

Everything about a person in one place — bio, contact info, tags, and full interaction history.

- Bio, personal notes, and birthday
- Flexible custom tags and categories
- Full timestamped interaction timeline

### 🔔 Reconnect Reminders

Never let a relationship fade quietly. KeenKeeper nudges you before it's been too long.

- Configurable reminder intervals per contact
- Priority contacts always visible at the top
- Snooze or mark as reconnected in one tap

### 🏷️ Tags & Categories

Organise your network your way — family, work, college, city — with unlimited custom tags.

- Create and assign unlimited tags
- Filter and search contacts by tag
- Bulk-edit tags across multiple contacts at once

### 📝 Interaction Log

Log calls, coffee chats, or messages with a quick note so you always have context for next time.

- Record date, interaction type, and free-text notes
- Searchable history across all contacts
- Quick-add log entry directly from a profile

## ⚙️ Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) v18+
- npm or yarn

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/your-username/keenkeeper.git

# 2. Navigate into the project
cd keenkeeper

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev
```

Open [http://keen-keeper-app.netlify.app](http://keen-keeper-app.netlify.app) in your browser.
